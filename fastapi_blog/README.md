In this we use uv for creating this project and using for pip
<p> To install uv we have to use a pip command to install uv, then after that create a virtual environment for every project and activate it.
<h5> pip install uv </h5>
<h5> python -m venv <i>nameofenv</i> </h5>
<h5> source <i>name</i>/bin/activate</h5> 
<p> Italic show the input space</p>

<h1>UV Commands</h1>
<p> To create a project <b>uv init <i>projectname</i></b></p>
<p> TO install the module in uv - <b>uv add <i>"fastapi[standard]"</i></b> </p>
<p> To run fastapi project using uv in developer - <b>uv run fastapi dev main.py</b>
